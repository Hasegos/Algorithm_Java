## 백준 알고리즘

백준 사이트를 참고한 알고리즘 및 풀이 저장소입니다.<br>

문제 풀이를 하면서 본인의 문제풀이 방법을 해설하고 배웠던 것을 기록하기위해 ```MarkDown```형태로 작성했습니다. <br>
직접 작성한 풀이와 다른 사람의 풀이 등 참고해서 구성했습니다. <br>
주로 JAVA 를 사용해서 문제를 풀었습니다. 참고 부탁드립니다! :) <br>
```
# 백준 문제 풀이 작성형식
./문제이름/README.md 
```
