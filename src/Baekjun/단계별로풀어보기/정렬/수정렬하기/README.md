## 문제

N개의 수가 주어졌을 때, 이를 오름차순으로 정렬하는 프로그램을 작성하시오.

___

## 입력

첫째 줄에 수의 개수 N(1 ≤ N ≤ 1,000)이 주어진다. 둘째 줄부터 N개의 줄에는 수가 주어진다.<br>
이 수는 절댓값이 1,000보다 작거나 같은 정수이다. 수는 중복되지 않는다.

---

## 출력

첫째 줄부터 N개의 줄에 오름차순으로 정렬한 결과를 한 줄에 하나씩 출력한다.

---

## 예제 입력 & 출력 1

|예제 입력| 예제 출력   |
|--|---------|
|5<br>5<br>2<br>3<br>4<br>1| 1<br>2<br>3<br>4<br>5 |

---

## 해결방안
**1.입력으로 주어진 n개의 정수를 배열에 저장한 뒤 오름차순으로 정렬하는 문제이다.**<br>
**2.정렬에는 자바 내장 메서드 Arrays.sort()를 사용하여 효율적인 정렬 알고리즘(O(n log n))을 수행한다.**<br>
**3.정렬이 완료된 배열의 원소를 순서대로 출력한다.**<br>
**4.`입력받기 → 정렬 → 출력`의 단계를 순차적으로 수행하여 간단히 해결할 수 있다.**
