## 문제

2022 연세대학교 미래캠퍼스 슬기로운 코딩생활에 `N`명의 학생들이 응시했다.

이들 중 점수가 가장 높은 `k`명은 상을 받을 것이다. 이 때, 상을 받는 커트라인이 몇 점인지 구하라.

커트라인이란 상을 받는 사람들 중 점수가 가장 가장 낮은 사람의 점수를 말한다.

___

## 입력

첫첫째 줄에는 응시자의 수 `N`과 상을 받는 사람의 수 `k`가 공백을 사이에 두고 주어진다.

둘째 줄에는 각 학생의 점수 `x`가 공백을 사이에 두고 주어진다.

---

## 출력

상을 받는 커트라인을 출력하라.

---

## 제한

+ 1 ≤ N ≤ 1000
+ 1 ≤ k ≤ N
+ 0 ≤ x ≤ 10000

---

## 예제 입력 & 출력 1

|예제 입력| 예제 출력 |
|--|-------|
|5 2<br>100 76 85 93 98| 98    |

---

## 해결방안
**1.지원자 수 `n`과 선발 인원 `k`, 점수 배열을 입력받아 점수를 오름차순으로 정렬한다.**<br>
**2.정렬 후 커트라인은 상위 `k`번째 점수이므로 0-기반 인덱스를 고려해 `score[n-k]`를 출력한다.**<br>
**3.정렬에는 `Arrays.sort`를 사용하므로 시간 복잡도는 O(n log n)이다.**<br>
**4.동점 처리는 정렬 결과에서 동일 점수가 그대로 유지되므로 추가 로직이 필요 없다.**
