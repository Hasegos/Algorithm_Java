## 문제

1번부터 N번까지 N개의 풍선이 원형으로 놓여 있고. i번 풍선의 오른쪽에는 i+1번 풍선이 있고, 왼쪽에는 i-1번 풍선이 있다.<br>
단, 1번 풍선의 왼쪽에 N번 풍선이 있고, N번 풍선의 오른쪽에 1번 풍선이 있다.<br>
각 풍선 안에는 종이가 하나 들어있고, 종이에는 -N보다 크거나 같고, N보다 작거나 같은 정수가 하나 적혀있다.<br>
이 풍선들을 다음과 같은 규칙으로 터뜨린다.

우선, 제일 처음에는 1번 풍선을 터뜨린다. 다음에는 풍선 안에 있는 종이를 꺼내어 그 종이에 적혀있는 값만큼 이동하여 다음 풍선을 터뜨린다.<br>
양수가 적혀 있을 경우에는 오른쪽으로, 음수가 적혀 있을 때는 왼쪽으로 이동한다. 이동할 때에는 이미 터진 풍선은 빼고 이동한다.

예를 들어 다섯 개의 풍선 안에 차례로 3, 2, 1, -3, -1이 적혀 있었다고 하자.<br>
이 경우 3이 적혀 있는 1번 풍선, -3이 적혀 있는 4번 풍선, -1이 적혀 있는 5번 풍선, 1이 적혀 있는 3번 풍선, 2가 적혀 있는 2번 풍선의 순서대로 터지게 된다.

___

## 입력

첫째 줄에 자연수 N(1 ≤ N ≤ 1,000)이 주어진다. 다음 줄에는 차례로 각 풍선 안의 종이에 적혀 있는 수가 주어진다. 종이에 0은 적혀있지 않다.

---

## 출력

첫째 줄에 터진 풍선의 번호를 차례로 나열한다.

---

## 예제 입력 & 출력 1

| 예제 입력            | 예제 출력     |
|------------------|-----------|
| 5<br>3 2 1 -3 -1 | 1 4 5 3 2 |

---

## 해결방안
**1.각 풍선의 번호와 이동 값을 덱에 저장하고, 가장 앞의 풍선을 터뜨리며 이동 값에 따라 덱을 회전시키는 방식으로 시뮬레이션한다.** <br>
**2.풍선을 터뜨릴 때 번호를 출력하고, 이동 값이 양수이면 (이동 값 − 1)만큼 앞의 요소를 뒤로 보내며 덱을 오른쪽으로 회전시킨다.** <br>
**3.이동 값이 음수이면 절댓값만큼 뒤의 요소를 앞으로 보내 덱을 왼쪽으로 회전시킨다.**<br>
**4.덱에 풍선이 하나 남을 때까지 반복한 뒤 마지막 남은 풍선의 번호를 출력하여 최종 순서를 완성한다.**
