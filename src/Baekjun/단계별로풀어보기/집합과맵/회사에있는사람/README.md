## 문제

상근이는 세계적인 소프트웨어 회사 기글에서 일한다.<br>
이 회사의 가장 큰 특징은 자유로운 출퇴근 시간이다. 따라서, 직원들은 반드시 9시부터 6시까지 회사에 있지 않아도 된다.

각 직원은 자기가 원할 때 출근할 수 있고, 아무때나 퇴근할 수 있다.

상근이는 모든 사람의 출입카드 시스템의 로그를 가지고 있다. 이 로그는 어떤 사람이 회사에 들어왔는지, 나갔는지가 기록되어져 있다.<br>
로그가 주어졌을 때, 현재 회사에 있는 모든 사람을 구하는 프로그램을 작성하시오.

___

## 입력

첫째 줄에 로그에 기록된 출입 기록의 수 n이 주어진다.<br>
(2 ≤ n ≤ 106) 다음 n개의 줄에는 출입 기록이 순서대로 주어지며, 각 사람의 이름이 주어지고 "enter"나 "leave"가 주어진다.<br>
"enter"인 경우는 출근, "leave"인 경우는 퇴근이다.

회사에는 동명이인이 없으며, 대소문자가 다른 경우에는 다른 이름이다. 사람들의 이름은 알파벳 대소문자로 구성된 5글자 이하의 문자열이다.

---

## 출력

현재 회사에 있는 사람의 이름을 사전 순의 역순으로 한 줄에 한 명씩 출력한다.

---

## 예제 입력 & 출력 1

| 예제 입력                                         | 예제 출력 |
|-----------------------------------------------|-------|
| 4<br>Baha enter<br>Askar enter<br>Baha leave<br>Artem enter | 4 |

---

## 해결방안
**1.출입 기록을 입력받아 ‘enter’일 경우 맵에 저장하고, ‘leave’일 경우 맵에서 해당 이름을 제거한다.**<br>
**2.모든 기록 처리 후 맵에 남아 있는 사람은 현재 회사에 있는 사람들이다.**<br>
**3.남은 이름들을 배열로 변환하여 사전 역순(내림차순)으로 정렬한다.**<br>
**4.정렬된 순서대로 출력해 회사에 남아 있는 사람들을 요구된 형식대로 출력한다.**
