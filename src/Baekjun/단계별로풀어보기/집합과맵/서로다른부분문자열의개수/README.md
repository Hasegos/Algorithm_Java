## 문제

문자열 S가 주어졌을 때, S의 서로 다른 부분 문자열의 개수를 구하는 프로그램을 작성하시오.

부분 문자열은 S에서 연속된 일부분을 말하며, 길이가 1보다 크거나 같아야 한다.

예를 들어, ababc의 부분 문자열은 a, b, a, b, c, ab, ba, ab, bc, aba, bab, abc, abab, babc, ababc가 있고, 서로 다른것의 개수는 12개이다.

___

## 입력

첫째 줄에 문자열 S가 주어진다. S는 알파벳 소문자로만 이루어져 있고, 길이는 1,000 이하이다.

---

## 출력

첫째 줄에 S의 서로 다른 부분 문자열의 개수를 출력한다.

---

## 예제 입력 & 출력 1

| 예제 입력                                         | 예제 출력 |
|-----------------------------------------------|-------|
| ababc | 12    |

---

## 해결방안
**1.문자열 s의 모든 부분문자열 s[i..j)를 이중 반복으로 생성한다.**<br>
**2.`HashSet`에 넣어 중복을 제거하고, 최종적으로 서로 다른 부분문자열의 개수를 `set.size()`로 구한다.**<br>
**3.부분문자열 생성은 총 O(n²)개이며, 자바의 `substring`이 복사 비용을 가지므로 전체 시간은 평균 O(n³) 수준이 될 수 있다.**<br>
**4.메모리는 서로 다른 부분문자열 수에 비례해 증가한다. 더 빠른 해법은 접미 자동자/접미 배열 같은 자료구조를 쓰는 방법이 있다.**
