## 문제

정수 n(0 ≤ n ≤ 4*10^9)가 주어졌을 때, n보다 크거나 같은 소수 중 가장 작은 소수 찾는 프로그램을 작성하시오.

___

## 입력

첫째 줄에 테스트 케이스의 개수가 주어진다. 각 테스트 케이스는 한 줄로 이루어져 있고, 정수 n이 주어진다.

---

## 출력

각각의 테스트 케이스에 대해서 n보다 크거나 같은 소수 중 가장 작은 소수를 한 줄에 하나씩 출력한다.

---

## 예제 입력 & 출력 1

| 예제 입력                     | 예제 출력          |
|---------------------------|----------------|
| 3<br>6<br>20<br>100 | 7<br>23<br>101 |

---

## 해결방안
**1.각 테스트 케이스마다 주어진 정수 n 이상인 수 중에서 가장 작은 소수를 찾아야 하므로, n부터 시작해 소수가 나올 때까지 1씩 증가시키며 검사한다.**<br>
**2.소수 판별 함수에서는 n ≤ 1이면 소수가 아니고, 2는 소수, 나머지 짝수는 소수가 아니라고 빠르게 걸러낸다.**<br>
**3.이후, 3부터 n의 제곱근(√n)까지 홀수만 2씩 증가시키며 나누어 떨어지는지 검사해 소수 여부를 판별한다.**<br>
**4.소수로 판별된 값을 `StringBuilder`에 누적하여 각 테스트 케이스마다 구한 `다음 소수`를 순서대로 출력한다.**
