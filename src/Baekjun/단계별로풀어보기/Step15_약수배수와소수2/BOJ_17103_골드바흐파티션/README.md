## 문제

+ 골드바흐의 추측: 2보다 큰 짝수는 두 소수의 합으로 나타낼 수 있다.

짝수 N을 두 소수의 합으로 나타내는 표현을 골드바흐 파티션이라고 한다. 짝수 N이 주어졌을 때, 골드바흐 파티션의 개수를 구해보자.<br>
두 소수의 순서만 다른 것은 같은 파티션이다.

___

## 입력

첫째 줄에 테스트 케이스의 개수 T (1 ≤ T ≤ 100)가 주어진다. 각 테스트 케이스는 한 줄로 이루어져 있고, 정수 N은 짝수이고, 2 < N ≤ 1,000,000을 만족한다.

---

## 출력

각각의 테스트 케이스마다 골드바흐 파티션의 수를 출력한다.

---

## 제한

+ 1 ≤ n ≤ 123,456

---

## 예제 입력 & 출력 1

| 예제 입력                          | 예제 출력          |
|--------------------------------|----------------|
| 5<br>6<br>8<br>10<br>12<br>100 | 1<br>1<br>2<br>1<br>6|

---

## 해결방안
**1.주어진 짝수 n을 두 소수의 합으로 표현하는 방법의 수를 구하는 문제이며, 이를 위해 에라토스테네스의 체로 소수를 미리 판별한다.**<br>
**2.2부터 n까지의 모든 수를 소수 후보로 초기화한 후, 배수를 제거하는 방식으로 소수가 아닌 수를 거른다.**<br>
**3.소수 판별이 완료되면 i와 n−i가 모두 소수인지 확인하며, i는 2부터 n/2까지만 검사해 중복 카운트를 방지한다.**<br>
**4.(i, n−i)가 소수 쌍이면 카운트를 증가시키고, 최종적으로 골드바흐 파티션의 개수를 출력한다.**
