## 문제

총 N개의 문자열로 이루어진 집합 S가 주어진다.

입력으로 주어지는 M개의 문자열 중에서 집합 S에 포함되어 있는 것이 총 몇 개인지 구하는 프로그램을 작성하시오.

___

## 입력

첫째 줄에 문자열의 개수 N과 M (1 ≤ N ≤ 10,000, 1 ≤ M ≤ 10,000)이 주어진다.

다음 N개의 줄에는 집합 S에 포함되어 있는 문자열들이 주어진다.

다음 M개의 줄에는 검사해야 하는 문자열들이 주어진다.

입력으로 주어지는 문자열은 알파벳 소문자로만 이루어져 있으며, 길이는 500을 넘지 않는다.<br>
집합 S에 같은 문자열이 여러 번 주어지는 경우는 없다.

---

## 출력

첫째 줄에 M개의 문자열 중에 총 몇 개가 집합 S에 포함되어 있는지 출력한다.

---

## 해결방안
**1.S에 포함된 문자열들을 해시맵에 저장하여 문자열 존재 여부를 O(1)로 조회할 수 있도록 한다.**<br>
**2.그 이후 Q개의 문자열을 입력받아 해시맵에 존재하는지 containsKey로 검사한다.**<br>
**3.존재하면 정답 카운트를 1 증가시키고, 존재하지 않으면 증가시키지 않는다.**<br>
**4.모든 문자열 검사 후 S에 포함된 문자열의 개수를 출력한다.**
