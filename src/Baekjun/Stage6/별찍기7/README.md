## 문제
예제를 보고 규칙을 유추한 뒤에 별을 찍어 보세요.
___
## 입력

첫째 줄에 N(1 ≤ N ≤ 100)이 주어진다.
___
## 출력

첫째 줄부터 2×N-1번째 줄까지 차례대로 별을 출력한다.
___

## 예제 입력 1

5

## 예제 출력 1

```
    *
   ***
  *****
 *******
*********
 *******
  *****
   ***
    *
```
---

## 해결방안
**1.먼저 찍히는 빈칸과 찍히는 별 개수를 파악** <br>
**2.반복문을 사용하여 홀수개 별과 순차적인 빈칸 수를 찍기** <br>
