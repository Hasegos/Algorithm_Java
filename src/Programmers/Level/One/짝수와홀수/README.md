## 문제

정수 num이 짝수일 경우 "Even"을 반환하고 홀수인 경우 "Odd"를 반환하는 함수, solution을 완성해주세요.

___

## 제한 사항

+ num은 int 범위의 정수입니다.
+ 0은 짝수입니다.

## 입출력 예

| num	 | return |
|------|--------|
| 3    | 	"Odd"  |
| 4    | "Even" |

___

## 해결방안
**1.짝수는 숫자 2로 나눈 나머지가 0일 경우 짝수이다.** <br>
**2.홀수는 숫자 2로 나눈 나머지가 1일 경우 홀수이다.** <br>
