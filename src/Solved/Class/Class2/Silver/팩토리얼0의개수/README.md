## 문제

N!에서 뒤에서부터 처음 0이 아닌 숫자가 나올 때까지 0의 개수를 구하는 프로그램을 작성하시오.

___

## 입력

첫째 줄에 N이 주어진다. (0 ≤ N ≤ 500)

___
## 출력

첫째 줄에 구한 0의 개수를 출력한다.

___

## 예제 입력 1

10

---

## 예제 출력 1

2

---
## 예제 입력 2

3

---

## 예제 출력 2

0

---

## 해결방안
**1.50!은 long 의 범위를 한참 뛰어넘는다. 그래서 곱해서 찾지말고 소인수분해로 찾아야한다.** <br>
**2.곱할 수들중에서 2와 5 둘다 존재할시 0이 생긴다.** <br>
