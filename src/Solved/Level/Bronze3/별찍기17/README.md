**1.(순서의 * 2 + 1) 개를 찍는데, 마지막줄을 제외한 나머지줄에서 각 끝에 두개를 제외해서는 없앤다.** <br>
**2.마지막줄은 정상적으로 마지막줄 * 2 +1 개를 찍는다.** <br>
